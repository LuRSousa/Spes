## Projeto Integrador: Spes

Este foi o meu primeiro projeto prático de desenvolvimento web, desenvolvido como parte do currículo escolar. O objetivo principal foi aplicar fundamentos de estruturação semântica em HTML5 e estilização personalizada com CSS3, consolidando o aprendizado inicial de front-end.

### Tecnologias
- **HTML5** (Estruturação semântica)
- **CSS3** (Estilização e Layout)

### Observação Técnica
*Este projeto foi desenvolvido com foco exclusivo para exibição em monitores **Desktop (1920x1080)**, não possuindo adaptação para dispositivos móveis (responsividade) por se tratar de um estudo inicial de layout fixo.*
